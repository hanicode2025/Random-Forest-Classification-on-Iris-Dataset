# Iris Dataset Random Forest Classifier

**Overview:**  
Random Forest classifier to predict Iris species using sepal and petal measurements. Optimized with GridSearchCV and evaluated with Accuracy & F1-score. Includes feature importance analysis.

**Technologies Used:**  
- Python 3.x  
- Pandas & NumPy  
- scikit-learn  
- Matplotlib & Seaborn  

**Dataset:**  
- Iris dataset (`iris.csv`) with features: sepal length, sepal width, petal length, petal width  
- Labels: Iris-setosa, Iris-versicolor, Iris-virginica  

**Project Workflow:**  
1. Data loading & preprocessing  
2. Model training with Random Forest  
3. Hyperparameter tuning using GridSearchCV  
4. Model evaluation (Accuracy & F1-score)  
5. Feature importance analysis  

**Key Skills Demonstrated:**  
- Classification with Random Forest  
- Hyperparameter tuning & model evaluation  
- Feature importance interpretation  
- Data preprocessing & visualization
